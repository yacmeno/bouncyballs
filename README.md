# bouncyballs: canvas game
## Overview
MDN offers assessments from their 'Learn web development' tutorials.
This bouncyballs repo is a version of this assessment: 
https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects/Adding_bouncing_balls_features

I added a timer to see the time it takes to capture all the balls.

## Play the game
Controls: WASD

Tested on chrome and firefox
```bash
git clone https://github.com/yacmeno/bouncyballs.git
cd path/to/repo
google-chrome index.html #or firefox index.html
```
